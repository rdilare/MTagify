# myMusic [MTagify]

Filter and List the songs from a database (json format) according to tags.

## dependencies 
```shell
pip3 install -r requirements_tui.txt
```

## run

```shell
python3 tui_app.py
```

## features
- filter songs by tags
- add tags to song
- delete tags from song
- view song info

## future features
- add new song in database
- update song info
- remove song from database
